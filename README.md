# LC Consultoria — TelecomKPI Manager

Plataforma SaaS premium para gestão de indicadores de Telecom.

## 🚀 Stack

- **Framework**: Next.js 14 (App Router)
- **Linguagem**: TypeScript
- **Styling**: Tailwind CSS
- **Ícones**: Lucide React
- **Deploy**: Vercel (região São Paulo — GRU)

## 🎨 Identidade Visual

- **Cores**: Preto (#0a0a0a) · Dourado (#d4af37) · Azul Marinho (#1a3a6b)
- **Tipografia Display**: Playfair Display
- **Tipografia Body**: DM Sans
- **Tipografia Mono**: JetBrains Mono

## 📁 Estrutura

```
src/
├── app/
│   ├── page.tsx          # Landing page
│   ├── login/page.tsx    # Página de login/cadastro
│   ├── dashboard/page.tsx # Dashboard (demo)
│   ├── layout.tsx        # Root layout
│   └── globals.css       # Estilos globais
├── components/           # Componentes reutilizáveis
└── lib/                  # Utilitários
public/
├── logo.svg              # Logo LC Consultoria
└── favicon.svg           # Favicon
```

## 🧑‍💻 Desenvolvimento

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
